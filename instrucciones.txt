instrucciones para ejecutarlo.

Debido a que es virtualmente imposible subir los 9000 prácticamente de archivos que hay en el directorio node_modules
(lo intenté, pero el programa se pegó y dió muchísimos errores), y como ya había un .gitignore que los ignoraba, 
los pasos para ejecutarlo son los siguientes:

Instalar nodejs desde la página.

entrar a la carpeta con cd <carpeta>

instalas las dependencias con:
npm install

para ejecutar, correr en la consola:
gulp

Luego copiar los archivos de este git y reemplazarlos por los otros.

Para la base de datos:
Se llama farmautomatico.
está en el archivo farmautomatico.sql, hay que importarla a "farmautomatico".
Para que funcione, el usuario es root y no tiene contraseña (que flojera).
Para importarla es entrando a mysql:
mysql -u root
luego crear la base de datos:
create database farmautomatico
entrar:
use farmautomatico
y llenarla con los datos:
source C:\Usuarios\Erick\farmautomatico1\farmautomatico.sql
(o la dirección donde esté, se arrastra a la terminal).

Con todo eso debería funcionar.
